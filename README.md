# MERN E-commerce Website with Personalized Product Recommendations - Prototype for Flipkart Grid 5.0

Welcome to the GitHub repository for our full-stack MERN e-commerce website with authentication, seller options, user options, and an admin dashboard. This project also features real-time payment processing, order status tracking, and personalized product recommendations. Our website serves as a prototype for Flipkart Grid 5.0, focusing on the problem statement of personalized product recommendations.

## Problem Statement - Personalized Product Recommendations

Our project aims to address the challenge of providing personalized product recommendations to users, enhancing their shopping experience. To achieve this, we have implemented the following features:

- Utilize a voice chatbot for user preferences and interactions.
- Construct user profiles with preferences and behavior insights.
- Develop an algorithm that combines preferences, popularity, and similarity to suggest products.
- Predict user-suited products using advanced recommendation algorithms.
- Define accuracy metrics to evaluate the quality of recommendations.
- Integrate personalized product ranking with popular items in the e-commerce domain.

## Features

- **Authentication:** Secure user authentication and registration process.
- **Seller Options:** Dedicated interface for sellers to manage their products and inventory.
- **User Options:** User accounts with order history, preferences, and profile management.
- **Admin Dashboard:** Admin panel to monitor and manage users, sellers, products, and orders.
- **Real-time Payment System:** Seamless payment processing for a smooth shopping experience.
- **Order Status Tracking:** Users can monitor the status and progress of their orders.
- **Personalized Product Recommendations:** Advanced recommendation engine for tailored suggestions.
- **Voice Chatbot:** Interactive voice-based chatbot for user engagement and preferences.
